Cố định nhân vật

Hướng dẫn tạo ảnh 2 (hay nhiều) nhân vật cố định cùng chung trong một khung hình

Kết hợp Character Reference và Vary - Region

Hiện nay Mid Journey vẫn chưa hỗ trợ chính thức tạo 2 (hay nhiều) nhân vật cố định, nhưng bạn có thể kết hợp sử dụng 2 tính năng Character Reference và Vary - Region để tạo nhiều nhân vật.

Cách làm

  • - Khởi tạo ảnh riêng 2 của nhân vật, sẽ có 2 giá trị thông số --cref URL character 1 và --cref URL character 2
  • - Tạo ảnh chung có 2 nhân vật, sử dụng ảnh nhân vật 1 để tạo --cref URL character 1
  • - Lúc này ảnh tạo ra sẽ gặp lỗi 2 nhân vật trong ảnh sẽ trộn lẫn, chọn ảnh tốt nhất và bật tính năng Vary - region, inpainting lại nhân vật lỗi với thông số --cref URL character 2

Nhân vật cố định thay đổi trang phục

Chức năng Nhân vật cố định Character Reference còn có thể giúp thay đổi Trang phục của nhân vật.

Bằng cách cho chỉ số --cw bằng 0 ( --cw 0 ), Character Reference sẽ chi tập trung vào khuôn mặt, giúp cho việc thay đổi trang phục.

/imagine prompt: <nhân vật> <trang phục> --cw 0 --cref URL character

Ví dụ: a man wearing a white blouse doctor --cw 0 --cref URLnhanvat

Nhân vật cố định thay đổi độ tuổi

/imagine prompt <nhân vật> <độ tuổi> --cw 0 --cref URL character

Example:

a baby, 5 years old --cw 0 --cref URL character

a teenage, 10 years old --cw 0 --cref URL character

a young boy, 25 years old --cw 0 --cref URL character

Tip: khi sử dụng tham số --cref có thể kết hợp nhiều URL hình ảnh nhân vật các độ tuổi để giữ sự cố định được tốt nhất, ví dụ: --cref URL nv1 URL nv2
